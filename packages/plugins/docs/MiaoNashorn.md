# MiaoNashorn

## 插件介绍

> 在Java14+环境下缺少Nashorn 本插件用于自动下载并且加载Nashorn依赖
> 自动从云端下载依赖 插件仅 8kb 大小

## 可用于下列插件 在 Java14+ 环境运行

- PlaceholderAPI 的 Javascript 扩展
- MiaoMenu
- MiaoScript
- TrMenu
- AttributePlus
- 等其他任何需要 Nashorn 引擎的插件
