## 下载 安装 插件

### 下载最新版本插件

- 插件商城: [下载地址](http://w.yumc.pw/free.html#MiaoScript-download)
- 构建站: [下载地址](https://ci.yumc.pw/job/Minecraft/job/MiaoScript/lastSuccessfulBuild/artifact/target/MiaoScript.jar)

### 安装到对应服务端的插件目录

- Spigot => plugins
- Sponge => mods
- BungeeCord => plugins
- Nukkit => plugins

### 重启服务器
